# Content Creator Skill

This skill generates compelling marketing content across all major formats and channels. It combines copywriting principles, SEO best practices, and platform-specific optimization to create content that drives engagement and conversions.

## When to Use This Skill

Activate this skill for:
- Blog posts and articles
- Social media posts (Instagram, LinkedIn, Twitter/X, Facebook, TikTok)
- Email campaigns and newsletters
- Landing page copy
- Video scripts and descriptions
- Ad copy (see ad-copywriter for paid ads specifically)
- Product descriptions
- Case studies and testimonials
- Lead magnets (ebooks, guides, checklists)
- Website copy

## Content Creation Framework

### 1. Understand the Context

Before writing, gather:
- **Purpose**: What is this content meant to achieve? (awareness, engagement, conversion)
- **Audience**: Who is this for? (use persona insights if available)
- **Platform**: Where will this be published? (blog, Instagram, email, etc.)
- **Tone**: What voice should this content have? (professional, casual, inspiring, educational)
- **SEO Keywords**: What terms should be incorporated naturally? (for web content)
- **Call-to-Action**: What should the reader do next?
- **Constraints**: Word count, character limits, brand guidelines

### 2. Apply Content Principles

**Engagement**:
- Hook readers in the first sentence/headline
- Use storytelling techniques
- Write in active voice
- Break up long paragraphs
- Use subheadings for scannability
- Include specific examples and data

**Conversion**:
- Focus on benefits, not features
- Address pain points and objections
- Include clear CTAs
- Create urgency when appropriate
- Use social proof and credibility markers

**Platform Optimization**:
- Match the platform's content style
- Respect character limits
- Use platform-specific features (hashtags, mentions, etc.)
- Optimize for how users consume content on that platform

### 3. SEO Best Practices (Web Content)

- Include primary keyword in title, first paragraph, and naturally throughout
- Use related keywords and semantic variations
- Write descriptive, keyword-rich meta descriptions
- Use header tags (H1, H2, H3) hierarchically
- Include internal and external links where relevant
- Optimize images with alt text
- Write for humans first, search engines second

## Content Type Guidelines

### Blog Posts & Articles

Structure:
```markdown
# Compelling Headline (Include Keyword)

**Meta Description**: [150-160 characters summarizing the post and including keyword]

[Engaging opening paragraph that hooks the reader and previews what's to come]

## Introduction
[Set up the problem/topic. Why should they care?]

## Main Section 1 (H2)
[Address first major point with examples and details]

### Subsection (H3 if needed)
[Break complex ideas into digestible chunks]

## Main Section 2 (H2)
[Address second major point]

## Main Section 3 (H2)
[Address third major point]

## Conclusion
[Summarize key takeaways]

**Call-to-Action**: [What should the reader do next?]
```

Best Practices:
- Aim for 1500-2500 words for in-depth posts
- Use bullet points and numbered lists
- Include images, charts, or visuals
- Add internal links to related content
- Include external links to authoritative sources
- Write scannable content with clear headings

### Social Media Posts

**Instagram**:
- Caption: 150-300 characters (short and punchy) or 1000+ (storytelling)
- Include line breaks for readability
- Use 5-10 relevant hashtags
- End with a question or CTA
- Describe visual content direction

```

[Main message with personality and value]

[Story or example that resonates]

[Call-to-action or question]

#hashtag1 #hashtag2 #hashtag3 #hashtag4 #hashtag5

```

**LinkedIn**:
- Professional but conversational tone
- Lead with a hook that shows up in the feed preview
- Use short paragraphs (1-2 sentences)
- Include data, insights, or lessons learned
- Tag relevant people/companies when appropriate

```
[Hook - strong statement or question]

[Expand on the hook with context]

[Share the main insight or story]

[Key takeaway or lesson]

[Call-to-action or question to drive engagement]

#RelevantHashtag #SecondHashtag
```

**Twitter/X**:
- 280 character limit, but aim for 100-150 for retweets
- One clear idea per tweet
- Use threads for longer narratives
- Include relevant hashtags (1-2 max)
- Add links when driving traffic

## Writing Best Practices

**Headlines That Convert**:
- Use numbers (5 Ways to..., 7 Steps to...)
- Ask questions (Why Do...?, How Can...?)
- Make promises (The Ultimate Guide to...)
- Create curiosity (The Secret to...)
- Include power words (Free, Proven, Easy, Fast, New)

**Calls-to-Action**:
- Be specific: "Download the Free Guide" not "Submit"
- Create urgency: "Get Your Free Audit Today"
- Focus on benefits: "Start Growing Your Business"
- Use first person: "Show Me How" not "Learn More"
- Make it visible and repeated 2-3 times in long content

**Common Mistakes to Avoid**:
- Don't bury the lede - lead with value
- Avoid jargon unless your audience uses it
- Don't make unsubstantiated claims
- Never write in passive voice
- Don't forget to proofread for errors
- Avoid walls of text - break it up

## Content Quality Checklist

Before delivering content, verify:
- [ ] Headline is compelling and includes keyword (for web content)
- [ ] Opening hook grabs attention immediately
- [ ] Content delivers on the headline's promise
- [ ] Benefits are clear and specific
- [ ] Tone matches the brand and platform
- [ ] CTAs are clear and action-oriented
- [ ] Content is scannable (subheadings, bullets, short paragraphs)
- [ ] No grammatical or spelling errors
- [ ] SEO elements included (for web content)
- [ ] Platform-specific requirements met (character counts, hashtags, etc.)
